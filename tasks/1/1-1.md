# 1-1: 투명 배경의 전체 화면 윈도우 ✅

## 목표
앱 윈도우를 전체 화면 크기로 만들고, 배경을 투명하게 설정

## 변경된 파일

### 1. `src-tauri/tauri.conf.json`
```json
{
  "maximized": true,      // ⭐ 추가: 창을 최대화 상태로 시작
  "decorations": false,   // 타이틀바, 테두리 없음
  "transparent": true,    // 배경 투명
  "alwaysOnTop": true,    // 항상 맨 위
  "skipTaskbar": true     // 작업표시줄에 안 보임
}
```
- `maximized: true` 추가로 전체 화면 차지
- `center: true` 제거 (maximized 상태에서 불필요)

### 2. `src/App.tsx`
```tsx
// Phaser 게임 크기를 전체 화면으로
width: window.innerWidth,
height: window.innerHeight,

// 창 크기 변경 대응
scale: {
  mode: Phaser.Scale.RESIZE,
  autoCenter: Phaser.Scale.CENTER_BOTH,
},

// 컨테이너도 전체 화면
style={{
  width: '100vw',
  height: '100vh',
  background: 'transparent',
}}
```

### 3. `src/game/PetScene.ts`
```ts
// 화면 중앙에 고양이 배치
const centerX = this.scale.width / 2
const centerY = this.scale.height / 2
this.pet = this.add.sprite(centerX, centerY, 'cat')

// 화면 리사이즈 대응
this.scale.on('resize', (gameSize) => {
  // 고양이가 화면 밖으로 나가면 안으로 이동
})
```

### 4. `src-tauri/CONFIG_NOTES.md` (새로 생성)
- JSON은 주석을 지원하지 않아서 설정 설명을 별도 문서로 작성

## 결과
- 윈도우가 전체 화면을 차지함
- 배경이 투명해서 고양이만 보임
- 고양이를 화면 어디든 드래그 가능
